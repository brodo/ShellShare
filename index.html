Welcome to TermShare!

TermShare allows you to share your shell with other people without the need to install anything!

On Linux: 
$$LINUX_COMMAND$$
On MacOs: 
$$MAC_COMMAND$$

Please press enter once again after running the command above! Don't type in passwords while using TermShare! TermShare does not work well with curses GUIs because 'script' doesn't. I do not log anything besides when a session starts, ends and the number of concurrent sessions.

TermShare should not be used in production and is a proof-of-concept for an interactive, zero-install console-oriented service.
